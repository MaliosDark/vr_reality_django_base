# Experiencia de Realidad Virtual con Controladores Oculus Rift en A-Frame

Este proyecto demuestra cómo integrar y manejar los controladores Oculus Rift en una experiencia de realidad virtual utilizando A-Frame.

## Descripción

El programa consiste en una experiencia de realidad virtual donde se muestran y se manejan los controladores de manos de los dispositivos Oculus Rift. Cada controlador representa una mano (izquierda y derecha) y puede detectar eventos de presionado del gatillo.

## Configuración

Para ejecutar este programa, sigue estos pasos:

1. Clona o descarga este repositorio en tu máquina local.
2. Abre el archivo `index.html` en un navegador web compatible con dispositivos de realidad virtual como Oculus Rift.

## Uso

La escena de realidad virtual mostrará dos controladores Oculus Rift, uno para cada mano. Cuando se presione el gatillo en cualquiera de los controladores, se mostrará un mensaje en la consola del navegador indicando qué gatillo fue presionado.

Puedes editar el archivo `index.html` para agregar lógica personalizada en JavaScript que maneje las acciones al presionar los gatillos.

## Requisitos

- Dispositivos Oculus Rift compatibles.
- Navegador web compatible con realidad virtual y soporte para WebXR.
